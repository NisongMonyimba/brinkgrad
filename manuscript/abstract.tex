We present \texttt{micro\-grad}, a complete open-source reference
implementation that integrates established techniques
(continuous adjoint, SUPG stabilisation, Brinkman--SIMP,
Helmholtz filtering, Heaviside projection, OC/MMA)
into a reproducible FEniCSx framework for
coupled flow-transport topology optimisation.
The contribution is the integration and documentation:
no equivalent open-source FEniCSx implementation
for coupled Brinkman--convection-diffusion
has been previously published.
The framework implements a multi-objective cost functional
coupling viscous dissipation and outlet concentration mismatch,
with SUPG-stabilised forward and adjoint transport
following standard practice~\cite{donea2003} at
$\Pe\sim10^2$--$10^5$,
and a modular continuation schedule reducing the gray-zone fraction
from $\grayShortRun$ to $\grayFrac$ over $\nIterPrimary$ iterations.

As a verification benchmark, the implementation achieves
RMSE~$=\rmseTopOpt$ for a linear concentration gradient
on an $80\times20$ mesh.
The linear target serves as a sanity check;
the hybrid OC/MMA schedule is validated by direct comparison
against pure MMA (RMSE~$=\rmsePureMMA$ vs.\ $\rmseShortRun$
at 600 iterations, Table~\ref{tab:mma_comparison}).
The benchmark is intended as a software verification case
rather than a mesh-independent design study;
mesh sensitivity results are reported in Section~\ref{sec:mesh_conv}.
\texttt{micrograd} optimises within the
Brinkman--SIMP porous-medium model;
the optimised permeability field is \emph{not} a manufacturable design.
Hard-thresholding to binary permeability increases RMSE by $+\binaryGapPct\%$,
confirming that the surrogate exploits gray-zone diffusion unavailable
in physical geometries.
Users requiring binary geometries should apply robust
projection~\cite{wang2011} during optimisation.
The complete pipeline reproduces in $\approx$1 hour
from a single Docker command
(Zenodo DOI:~\href{https://doi.org/10.5281/zenodo.20479523}{10.5281/zenodo.20479523},~\cite{micrograd2025zenodo}).
